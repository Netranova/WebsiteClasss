Instalasi dan Konfigurasi Sistem Operasi

Persiapan instalasi sistem operasi:

Menyiapkan file ISO dari sistem operasi (contoh: Windows 10), Membuat media instalasi bootable menggunakan software seperti Rufus, Menentukan jenis partisi (MBR atau GPT) sesuai perangkat target.

Pengaturan BIOS/UEFI:

Mengatur boot priority agar flashdisk terbaca lebih dulu, Menonaktifkan fitur Secure Boot jika instalasi tidak kompatibel, Mengaktifkan Legacy Mode atau UEFI sesuai kebutuhan.

Langkah instalasi sistem operasi:

Memilih drive target dan melakukan partisi atau format, Menyelesaikan wizard instalasi OS hingga reboot otomatis, Pengaturan awal sistem: nama perangkat, zona waktu, dan user account.

Verifikasi pasca instalasi:

Mengecek fungsi dasar seperti respons sistem, keyboard, dan layar, Memastikan sistem tidak mengalami crash saat startup, Melakukan pengecekan terhadap lisensi sistem (aktivasi).

Instalasi dan Update Driver Perangkat

Pendeteksian perangkat keras tanpa driver:

Menggunakan Device Manager untuk mendeteksi perangkat yang belum dikenali, Mengidentifikasi hardware ID sebagai acuan pencarian driver.

Instalasi driver secara manual:

Mengunduh driver dari situs resmi vendor perangkat (misal: intel.com, asus.com), Melakukan instalasi satu per satu: VGA, audio, LAN, WLAN, chipset, dan touchpad.

Instalasi driver secara otomatis:

Menggunakan software seperti DriverPack Solution atau Snappy Driver Installer, Menjalankan pemindaian otomatis dan pemasangan semua driver secara batch.

Verifikasi fungsi driver:

Memastikan koneksi jaringan dapat digunakan setelah driver LAN/WLAN terpasang, Mengatur resolusi layar tinggi setelah VGA driver diinstal, Menguji suara speaker/headset setelah driver audio terpasang.

Instalasi dan Pengaturan Aplikasi Pendukung Sistem

Instalasi aplikasi perkantoran:

Microsoft Office (Word, Excel, PowerPoint), Pengaturan aktivasi dan update otomatis.

Instalasi aplikasi utilitas:

WinRAR untuk kompresi dan ekstraksi file arsip, Google Chrome atau Mozilla Firefox sebagai browser utama, VLC Media Player untuk pemutaran file audio dan video.

Aplikasi pengelola sistem:

CCleaner untuk pembersihan sistem dan registry, Task Manager untuk mengatur aplikasi startup, Autoruns untuk menonaktifkan layanan tidak penting.

Instalasi antivirus dan keamanan:

Antivirus lokal (Smadav) dan/atau global (Avast, Bitdefender), Pemindaian awal setelah instalasi dan aktivasi fitur proteksi real-time.

Penghapusan bloatware:

Menghapus aplikasi bawaan sistem yang tidak diperlukan, Mengurangi beban RAM dan mempercepat booting sistem.

Perawatan, Pembersihan, dan Penanganan Perangkat Keras

Pembersihan fisik internal perangkat:

Membongkar casing laptop/PC menggunakan obeng presisi, Membersihkan motherboard, fan, dan heatsink dengan blower atau kuas halus, Mengganti thermal paste pada prosesor untuk pendinginan optimal.

Pemeriksaan fungsi komponen:

Menjalankan fan test untuk memastikan kipas berjalan normal, Mengecek kondisi baterai, RAM, dan konektor power menggunakan multimeter.

Perawatan port dan slot:

Membersihkan port USB, HDMI, LAN, dan jack audio menggunakan cairan isopropil alkohol, Memastikan tidak ada port yang longgar atau patah.

Pembersihan eksternal perangkat:

Membersihkan keyboard dari debu menggunakan vacuum mini, Mengelap layar LCD dengan lap microfiber, Membersihkan body laptop/PC dengan cairan pembersih non-korosif.
