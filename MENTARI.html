Penginstalan dan Pemulihan Sistem Operasi Windows

Kegiatan penting yang dilakukan adalah menginstal ulang sistem operasi, terutama Windows 10, menggunakan software backup recovery Acronis True Image 2019 Build 17750. Berikut tahapan teknisnya:

Persiapan Instalasi: Media instalasi (flashdisk/DVD) dimasukkan ke laptop/PC. Komputer dinyalakan, lalu pengguna masuk ke BIOS untuk mengatur urutan boot agar flashdisk berada di urutan teratas.

Proses Instalasi: Setelah BIOS tersetting dan proses boot berjalan, pengguna memilih menu seperti Program and Wipe, kemudian memilih opsi Backup and Restore di Acronis.

Recovery Image: Proses pemulihan dilakukan dengan memilih “Disk and Partition Recovery” untuk mengatur partisi yang akan dipulihkan. Jika partisi D menyimpan data penting, hanya partisi C yang dipilih agar data tetap aman.

Restart dan Driver Installation: Setelah proses recovery selesai, sistem akan restart otomatis. Saat itu, media instalasi dicabut. Proses selanjutnya adalah menginstal driver dengan mencolokkan kembali harddisk eksternal dan memastikan antivirus dinonaktifkan terlebih dahulu.

Kegiatan ini memberikan pembelajaran teknis lengkap dalam pemulihan sistem berbasis image, manajemen BIOS, dan instalasi driver setelah OS selesai dipulihkan.

Instalasi dan Pengaturan Aplikasi Perkantoran (Microsoft Office)

Setelah sistem operasi diinstal ulang, penggunaan dan penginstalan Microsoft Office menjadi bagian penting dari aktivitas PKL. Tahapan instalasinya antara lain:

Aktivasi Windows 11: Proses dimulai dengan menghidupkan laptop dan melakukan aktivasi Windows yang telah diinstal.

Koneksi Internet & Akun Microsoft: Siswa menghubungkan laptop ke Wi-Fi, melakukan login dengan akun Gmail, dan membuat PIN untuk keamanan.

Update Windows: Sistem akan melakukan update otomatis, memakan waktu sekitar 30 menit – 1 jam.

Instalasi Office: Setelah update, Microsoft Office seperti Word, Excel, PowerPoint, dan Access diinstal.

Personalisasi Sistem: Melalui Control Panel, dilakukan pengaturan seperti tampilan, tema, pemisahan drive (partisi C dan D), dan pengaturan waktu.

Manajemen Penyimpanan: Menggunakan fitur ‘Manage’ pada This PC untuk mempartisi penyimpanan serta mengelola keamanan sistem.

Bagian ini memperkenalkan peserta PKL pada pengelolaan dasar-dasar sistem operasi modern, software produktivitas, serta personalisasi dan konfigurasi sistem Windows.

Instalasi dan Penggunaan Software Pendukung

Selain Office, siswa juga dikenalkan dengan berbagai software aplikasi yang mendukung kegiatan kantor, multimedia, keamanan, dan pengelolaan data. Beberapa software yang dipelajari:

Google Chrome & Mozilla Firefox: Browser untuk keperluan pencarian dan koneksi internet.

Microsoft Word, Excel, PowerPoint: Aplikasi utama pengolah dokumen dan data.

VLC & KMPlayer: Pemutar multimedia untuk berbagai format video/audio.

Winamp: Pemutar musik.

Nitro Pro: Untuk pengelolaan file PDF (menggabung, edit, format).

Photoshop & CorelDraw: Untuk editing gambar dan desain grafis 2D (vektor/ilustrasi).

AVG Protection: Antivirus untuk perlindungan dari malware dan ancaman keamanan lainnya.

UltraISO: Untuk membuat salinan data CD/DVD seperti aslinya.

Recycle Bin & Control Panel: Untuk manajemen file dan sistem konfigurasi.

Siswa tidak hanya diajari cara menginstal, tapi juga fungsi dari masing-masing software tersebut. Kegiatan ini memperluas kemampuan teknis dalam manajemen dan penggunaan software aplikasi dalam dunia kerja nyata.

Praktik Servis dan Pemeliharaan Perangkat Keras

Peserta PKL melakukan berbagai kegiatan pemeliharaan perangkat keras, termasuk:

Mengisi Tinta Printer: Praktik pengisian tinta secara manual dan cara mereset printer setelah pengisian.

Instalasi Driver Printer: Melakukan instalasi driver khusus (misalnya Epson L3210) dan melakukan pengujian (test print) untuk memastikan hasil cetakan sesuai.

Pembongkaran Printer & Laptop: Mengamati dan melakukan pembongkaran perangkat keras seperti laptop dan printer untuk mengetahui susunan dan komponen internal.

Instalasi Aplikasi Pendukung: Setelah pembongkaran, dilakukan pengujian software untuk melihat kestabilan dan kinerja perangkat.

Melalui aktivitas ini, siswa memperoleh pengetahuan dasar hingga menengah dalam troubleshooting perangkat keras, mengasah keterampilan teknis, serta prosedur keselamatan kerja saat menangani hardware elektronik.
