Perakitan dan Perbaikan Perangkat Keras

Materi ini mencakup berbagai kegiatan teknis pada level perangkat keras, termasuk:

Pembongkaran dan Pemasangan Komponen Laptop:

Mencakup proses membuka casing, melepas dan memasang ulang hardware seperti keyboard, LCD, mainboard, RAM, harddisk, hingga baterai CMOS. Langkah ini penting untuk mengenali letak dan fungsi tiap komponen serta teknik servis standar.

Penggantian Komponen:

Meliputi aktivitas mengganti hardware yang rusak atau usang, seperti mengganti harddisk dengan SSD, mengganti RAM untuk peningkatan performa, mengganti LCD yang retak, atau mengganti mainboard yang rusak.

Perakitan Unit Komputer:

Materi ini menjelaskan tentang merakit CPU dari awal, menyusun komponen seperti prosesor, motherboard, PSU, HDD/SSD, dan kipas pendingin ke dalam casing, hingga melakukan pengkabelan internal.

Quality Control (QC):

Setelah perakitan dan perbaikan, dilakukan pengecekan kelayakan fungsi perangkat. Kegiatan ini meliputi uji performa perangkat, pengujian kestabilan sistem, pengecekan konektor, dan identifikasi cacat teknis.

Instalasi dan Konfigurasi Sistem Operasi serta Software Pendukung

Materi ini berisi kegiatan instalasi sistem operasi dan aplikasi tambahan, antara lain:

Pembuatan Media Bootable (Bootingan):

Menggunakan tools seperti Rufus untuk membuat USB installer sistem operasi (Windows 7, 8.1, atau 10). Ini mencakup proses formatting flashdisk dan konfigurasi ISO OS.

Pengaturan BIOS dan UEFI:

Materi ini mencakup cara mengakses BIOS/UEFI, mengganti prioritas boot, mengaktifkan/menonaktifkan fitur secure boot, serta mengenali antarmuka sistem firmware.

Instalasi Sistem Operasi Windows:

Proses mencakup partisi harddisk, pemilihan drive instalasi, instalasi driver dasar, dan pengaturan awal sistem (akun pengguna, bahasa, zona waktu, dan aktivasi OS).

Instalasi Driver dan Aplikasi:

Setelah instalasi OS, dilakukan pemasangan driver chipset, VGA, LAN/WiFi, serta aplikasi dasar seperti Microsoft Office, WinRAR, browser, antivirus, dan program utilities lainnya.

Konfigurasi dan Pengelolaan Jaringan Komputer

Materi ini berkaitan dengan pembuatan dan pengelolaan jaringan menggunakan perangkat Mikrotik, termasuk:

Konfigurasi Dasar Mikrotik:

Menyetel IP address, DNS, NAT, DHCP server, dan bridge interface. Ini merupakan fondasi dari layanan jaringan berbasis router.

Hotspot Management:

Mencakup pembuatan jaringan hotspot menggunakan Mikrotik RouterOS. Pengaturan user hotspot dilakukan melalui User Profile, IP binding, dan session timeout.

Bandwidth Management:

Teknik membatasi kecepatan akses internet berdasarkan user, interface, atau IP tertentu dengan menggunakan fitur queue simple atau queue tree.

Blokir Situs Web dan Filtering Konten:

Menggunakan fitur firewall filter rules pada Mikrotik untuk memblokir situs-situs tertentu yang tidak diizinkan.

Virtual LAN (VLAN):

Materi tambahan pada beberapa siswa mencakup segmentasi jaringan menggunakan VLAN pada Mikrotik, yang digunakan untuk membagi jaringan berdasarkan fungsi atau departemen.

Troubleshooting dan Maintenance Perangkat

Materi ini memuat metode pemeliharaan dan penanganan masalah teknis komputer dan laptop:

Diagnosa Masalah Perangkat:

Mengidentifikasi penyebab laptop tidak menyala, hang, lambat, atau overheating. Pengujian dilakukan dengan mengganti komponen secara selektif atau menjalankan tools diagnostik.

Pembersihan dan Perawatan Hardware:

fisik pada kipas, motherboard, dan slot RAM menggunakan kuas, blower, atau alkohol isopropil. Langkah ini penting untuk mencegah overheating dan konsleting.

Backup dan Recovery Data:

Mencakup pencadangan data sebelum instalasi ulang sistem atau penggantian harddisk. Tools seperti AOMEI, EaseUS, atau fitur bawaan Windows digunakan untuk cloning atau backup.

Pengujian Kesehatan Harddisk dan Baterai:

Menggunakan aplikasi seperti CrystalDiskInfo, HDTune, dan BIOS diagnostics untuk memeriksa kesehatan fisik storage dan baterai laptop.
